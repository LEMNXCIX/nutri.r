# nutri.r brand assets

- `app-icon.svg`: canonical app icon used to generate the favicon and Tauri icon set.
- `wordmark.svg`: pill lockup for presentations and documentation.
- `wordmark-light.svg`: transparent lockup for light surfaces.
- `wordmark-dark.svg`: transparent lockup for dark surfaces.

Color palette:

- `#0A0A0A` brand black
- `#FFFFFF` brand white
- `#0DF259` brand green
